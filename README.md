# Sales Data Analysis Dashboard (Power BI)

## 📊 Project Overview

This project is a data analysis dashboard created using Microsoft Power BI.
The dashboard analyzes sales data to identify trends, regional performance, and top-performing products.

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-selling products
* Compare sales across regions
* Visualize monthly sales trends

## 🛠 Tools Used

* Microsoft Power BI
* Microsoft Excel (Dataset)
* Data Visualization Techniques

## 📂 Dataset

The dataset contains sales information such as:

* Order Date
* Region
* Product Category
* Sales Amount
* Profit

## 📈 Dashboard Features

* Sales by Region
* Monthly Sales Trend
* Top 10 Products by Sales
* Profit Analysis
* Category-wise Sales Distribution

## 🔍 Key Insights

* The West region generated the highest sales.
* Technology products produced the highest profit.
* Sales increased significantly during the holiday season.
